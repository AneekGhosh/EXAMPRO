# 🚀 ExamPro

A modern online examination and assessment platform built using Spring Boot, Spring Security, Thymeleaf, JPA, and H2 Database.

ExamPro provides a secure environment for administrators to create and manage examinations while allowing students to take exams, review results, and track their performance.

## ✨ Features

### Admin Features

### Student Features

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/AneekGhosh/EXAMPRO.git
cd EXAMPRO
```

### Run Application

```bash
mvn spring-boot:run
```

Application will start at:

```text
http://localhost:7890
```
